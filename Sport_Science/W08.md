# 收縮
- 向心收縮
    - 肌肉縮短
- 離心收縮
    - 肌肉伸長
- 等長收縮
    - 發力時，肌肉長度不變
    - 較省力
    - 同強度下需要更多的張力，肌肉會變大
- 等張收縮
    - 發力時，肌肉張力不變
- 等速收縮
    - 發力時，肌肉速度不變

# 能量
- 攝入：吃
- 輸出
    - 基礎代謝， 一般健康成人消耗60%～75%
    - 消化時的消耗，10%
    - 生活型態與運動，15%～80%
- 吃>輸出
    - 能量正平衡
    - 7700 kcal = 1kg

# 運動生理學
- 短期、長期適應
- 恆定（平時）與穩定（運動時心跳的穩定狀態，常運動會較低）
